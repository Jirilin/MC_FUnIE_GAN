# MC-FUnIE-GAN: Multi-Scale Convolutional Fast Underwater Image Enhancement GAN
MC-FUnIE-GAN integrates FUnIE-GAN generator architecture to capture contectual information across varying respective fields

## Overview
This repository contains the implementation of MC-FUnIE-GAN for underwater image enhancement.

## Features
- Multi-Scale Dilated Convolution (MSDC) module
- Hybrid Gannet-Lion Optimization (HGLO)
- Support for UIEB and UTIEB datasets
- Comprehensive evaluation metrics (PSNR, SSIM, UIQM, UICM, UISM, UIConM)
